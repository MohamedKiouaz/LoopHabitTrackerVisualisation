# LoopHabitTrackerVisualisation

Create visualisation of your Habits.

## How to

 1. Export your database as CSV.
 2. Unzip the output.
 3. Copy the Checkmark.csv next to the main.py
 4. Install the python packages in requirements.txt
 3. Run the script.
